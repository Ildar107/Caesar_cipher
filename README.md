# Caesar_cipher


### Установка NPM модулей

```
npm install
```

### Запуск приложения

```
node ./bin/index.js
```


### Доступные команды

```
-a --action : одно из действий decode/encode
-s --shift: размер смещения алфавита
-i --input: путь к входному файлу
-o --output: путь к выходному файлу
```
